Integrates the Petals Exchange with Gravity Forms

== Description ==

Send orders via the Petals Exchange using a [Gravity Forms](https://rocketgenius.pxf.io/bakkbone) form. _(affiliate link)_

**Recommended for non-customer-facing forms only as this integration transmits orders _directly_ to Petals Network.**

== Installation ==

= Automatic installation =
1. Search for "Petals Exchange" in the Plugin Repository from the Plugins > Add New screen
= Manual installation =
1. Unzip the plugin archive on your computer
2. Upload `relay-sender-petals-exchange-for-gravity-forms` directory to your `/wp-content/plugins/` directory
3. Activate the plugin through the `Plugins` menu in WordPress

== Frequently Asked Questions ==
**What plugins do I need?**
Required:

* [Gravity Forms](https://rocketgenius.pxf.io/bakkbone) (affiliate link)

**Does this plugin use any external services?**
Yes - the one that this plugin solely exists to connect you to. That's its whole purpose.
Orders are transmitted via the [Petals Exchange API](https://www.petals-exchange.com/relay/transmission/inbound.htm).
